# Simple Addition Kata

Write a function that adds two numbers together.

## Requirements

1. The function should be named `add`
2. It should take two parameters
3. It should return the sum of the two parameters
4. It should work with both integers and floating point numbers

## Examples

- add(1, 2) should return 3
- add(1.5, 2.5) should return 4.0
- add(-1, 1) should return 0
